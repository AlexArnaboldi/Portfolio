SHADOWLOOP

First-person puzzle horror game

Shadowloop is a first-person horror puzzle game in which the player must solve environmental challenges to break out of a recurring time loop.

The project was developed in a structured team environment, simulating a professional studio workflow with a clear division between designers and programmers. The game features interactive physics-based object rotation, enemy AI, and dynamic level transitions.

Role: Game Programmer.
  • Unreal Engine 5
  • C++
  • Blueprint

My contributions:

  • Developed the core loop mechanic, repositioning the player in real time to simulate a recursive environment.
  • Created the main menu system, including communication between UI elements and in-scene Actors for a responsive startup flow.
  • Designed and implemented the in-game UI: pause menu, game over screen, and credits panel, building a communicative flow between transitions and menus to maintain clarity and immersion.
  • Built a basic save system to store key player progress and game state data across sessions.
